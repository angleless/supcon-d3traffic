|   bqueues   |   反向队长  |       array       | [37.9, 20.9, 15.9, 26.8, 59.8] |
|   btotals   |   反向路长  |       array       | [340, 498, 491, 298, 166] |
|    width    |   视口宽度  |       number      | 500 |
